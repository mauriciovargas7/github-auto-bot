## 🔐 Configuração do Git (Git Bash)

Para que o script funcione corretamente (principalmente o `git push` automático), é necessário configurar o Git no seu ambiente.

---

### 🟢 1. Configurar usuário no Git

Abra o Git Bash e execute:

```bash
git config --global user.name "SEU_NOME"
git config --global user.email "SEU_EMAIL"
